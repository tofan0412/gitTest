1. charAt / charCodeAt

charAt(index) : index 위치의 문자를 리턴한다.
charCodeAt(index) : index 위치의 유니코드 값을 리턴한다.



2. indexOf / lastIndexOf

str = "가나다/사자/호랑이/고양이/치타/코끼리/표범/사자/가나다";
indexOf("문자열") : 문자열의 위치를 검색한다.
lastIndexOf("문자열") : 문자열의 위치를 검색 - 뒤에서부터 시작한다.



3. concat/trim

str2 = "사과/복숭아/자두/참외/수박/바나나";
concat() : 두 개 이상의 문자열을 서로 합친다.
str4 = "      사과 바나나 복숭아 딸기 바나나    "
trim() : 문자열 앞,뒤의 공백을 제거한다.



4. substring/slice/substr

str = "가나다/사자/호랑이/고양이/치타/코끼리/표범/사자/가나다";
substring(start, end) : start ~ end-1 까지의 문자열을 추출한다.
slice(start, end) : start ~ end-1까지의 문자열을 추출한다.
substr(start, length) : start ~ length 갯수를 추출한다.



5. split

str = "가나다/사자/호랑이/고양이/치타/코끼리/표범/사자/가나다";
split(구분자, 개수) : 구분자로 문자를 구분해서 배열로 리턴한다.



6. replace

str = "가나다/사자/호랑이/고양이/치타/코끼리/표범/사자/가나다";
replace(old, new) : old의 값을 new의 값으로 변경한다.
replace(//g) : /와 / 사이에 정규 표현식을 사용할 수 있다, g는 global의 약자로, old에 해당하는 모든 값을 new로 바꾸는 것을 의미한다.
