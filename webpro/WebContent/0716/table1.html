작성팁
 table : 표 형태의 데이터를 표시한다. 
 tr : table row(행 또는 줄을 의미한다.)
 td : table data(칸을 의미한다.)
 th : 표의 제목을 의미하며 자동으로 텍스트가 가운데 정렬이 되고, 글씨가 bold처리 된다. 
 
 스타일
 table : border, border-collapse
 td : width, height, text-align 
 tr : background.
 th : width , height
