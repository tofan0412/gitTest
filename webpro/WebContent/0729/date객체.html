1. Date 객체의 생성

- new Date()
- new Date(12345678) => 1970년 1월 1일을 기준으로 얼마나 지났는지를 출력해준다.
- new Date("2010.02.23")
- new Date(2000,01,23,12,30,12)
- 월은 0월부터 시작한다. 숫자로 Date() 객체를 생성할 때, 하나 작은 값으로 설정된다.


2. Date 객체의 생성

- now = new Date()
- now.toLocaleString() : 날짜와 시간
- now.toLocaleDateString() : 날짜
- now.toLocaleTimeString() : 시간



3. 요일 구하기

- now = new Date()
- day = now.getDay()
